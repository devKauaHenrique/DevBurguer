# 🍔 Dev Burguer

Uma landing page moderna para hamburgueria, desenvolvida com HTML, Tailwind CSS e JavaScript, focada em apresentar produtos, destacar promoções e facilitar o contato com os clientes.

## 🚀 Demonstração

🔗 Acesse o projeto online:

https://dev-burguer-seven.vercel.app

## ✨ Funcionalidades

* Apresentação dos produtos da hamburgueria
* Cardápio visual e atrativo
* Design responsivo para dispositivos móveis
* Integração com localização da hamburgueria
* Botão de contato rápido
* Animações para melhorar a experiência do usuário

## 💻 Como Executar Localmente

1. Clone o repositório:

```bash
git clone https://github.com/devKauaHenrique/DevBurguer.git
```

2. Entre na pasta:

```bash
cd DevBurguer
```

3. Instale as dependências:

```bash
npm install
```

4. Execute o projeto:

```bash
npm run dev
```
